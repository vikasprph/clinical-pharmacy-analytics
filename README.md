# Clinical Pharmacy Cost-Savings Analysis

## Overview
This project analyzes pharmacy claims data to identify high-cost drug utilization and potential cost-saving opportunities using clinical and data analytics approaches.

## Tools Used
- Python (pandas)
- SQL (SQLite)
- Tableau Public

## Key Insights
- Brand medications accounted for the majority of total pharmacy spend
- Specialty drugs were the primary cost drivers
- Significant savings opportunities identified through:
  - Generic substitution
  - Biosimilar adoption
  - Therapy optimization

## Files
- pharmacy_claims_final.csv → dataset
- claims_analysis.sql → SQL queries
- dashboard.png → Tableau visualization
